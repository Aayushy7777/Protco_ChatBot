┌──────────────────────────────────────────────────────────────────────────────┐
│ STEP 1: START SERVICES                                                       │
└──────────────────────────────────────────────────────────────────────────────┘

Option A - Start Everything at Once:
  1. Double-click: START_ALL.bat
  2. Wait 10 seconds for services to start
  3. Browser opens to http://localhost:3000

Option B - Manual Start (if you want to see logs):
  Terminal 1: ollama serve
  Terminal 2: cd ai-dashboard-generator\backend && start-backend.bat
  Terminal 3: cd mission-control && pnpm dev

┌──────────────────────────────────────────────────────────────────────────────┐
│ STEP 2: CHECK STATUS                                                         │
└──────────────────────────────────────────────────────────────────────────────┘

In Mission Control (http://localhost:3000):
  ✓ Go to "CSV Chat Dashboard" panel
  ✓ You should see 3 status indicators:
    🟢 Backend   = Connected to http://localhost:8010
    🟢 Ollama    = Connected to http://localhost:11434  
    🟢 File      = Ready to chat

If any are red:
  ❌ Backend red?   → Run START_ALL.bat or START_OLLAMA_AND_BACKEND.bat
  ❌ Ollama red?    → Open terminal and run: ollama serve
  ❌ File red?      → Continue to Step 3

┌──────────────────────────────────────────────────────────────────────────────┐
│ STEP 3: UPLOAD FILE                                                          │
└──────────────────────────────────────────────────────────────────────────────┘

  1. Click "📤 Upload File" button
  2. Select a CSV or Excel file
  3. Wait for upload to complete
  4. File status should show green ✓

┌──────────────────────────────────────────────────────────────────────────────┐
│ STEP 4: CHAT!                                                                │
└──────────────────────────────────────────────────────────────────────────────┘

Once all 3 status lights are green:
  • Type a question in the chat box
  • Press Send or Enter
  • Get instant answers about your data!

Example Questions:
  "What columns do I have?"
  "What is the total revenue?"
  "Show me the top 5 products"
  "What are the trends?"
  "Find outliers in the data"

║                          TROUBLESHOOTING                                     ║
╚══════════════════════════════════════════════════════════════════════════════╝

PROBLEM: Backend shows as disconnected (red)
SOLUTION: 
  • Make sure START_OLLAMA_AND_BACKEND.bat finished starting
  • Check if http://localhost:8010/api/health works in browser
  • If not, close any process on port 8010 and restart

PROBLEM: Ollama shows as disconnected (red)
SOLUTION:
  • Start Ollama in a new terminal: ollama serve
  • Make sure it says "Listening on 127.0.0.1:11434"
  • Try pulling the model: ollama pull llama3.1

PROBLEM: File status shows grey (no file)
SOLUTION:
  • Click "📤 Upload File" button in the panel
  • Select a CSV or Excel file from your computer
  • Wait for the upload to complete

PROBLEM: Chat is slow/not responding
SOLUTION:
  • First chat is always slower (model is processing)
  • Check that all 3 status lights are GREEN
  • Make sure your computer has enough RAM free
  • Try asking a simpler question first

PROBLEM: Getting errors about missing dependencies
SOLUTION:
  • Delete ai-dashboard-generator/backend/.venv folder
  • Run START_OLLAMA_AND_BACKEND.bat again
  • It will recreate the environment
